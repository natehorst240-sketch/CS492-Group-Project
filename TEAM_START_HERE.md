# Team Start Here

## One-time setup

1. Accept the GitHub collaborator invitation.
2. Install GitHub Desktop.
3. In GitHub Desktop, choose **File → Clone repository**.
4. Select `AceReads-Team-Project`.
5. Follow the setup steps in `README.md`.

## Every time you work

1. Open GitHub Desktop.
2. Click **Fetch origin**, then **Pull origin** if it appears.
3. Create a branch named with your task and name, such as `T2-003-kyonna`.
4. Open only your pre-labeled task file.
5. Paste or write your code below `PASTE YOUR CODE BELOW`.
6. Test it.
7. In GitHub Desktop, enter a summary beginning with the task ID.
8. Click **Commit**, then **Push origin**.
9. Tell the group in Discord that the branch is ready.

## Do not do these things

- Do not share database passwords in Discord or GitHub.
- Do not edit another person's task file without telling them.
- Do not put source code in Word documents.
- Do not download and email separate project copies.
- Do not push directly to `main` unless the team has agreed.
