# Contributing

Use one branch per task:

```text
T1-001-ali
T2-003-kyonna
T2-008-nathan
```

Commit messages should begin with the task ID:

```text
T2-008 add order status validation
T2-009 fix order total display
```

Before merging:

- The application starts without an error.
- No password or `.env` file is included.
- The assigned task ID appears in the filename and commit.
- Another teammate has reviewed the change.
