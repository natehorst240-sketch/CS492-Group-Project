# Shared Project Documents

## Sprint planning workbook

`CS492_Sprint_Planning.xlsx` is the group Sprint Planning document.

### Safe editing workflow

Excel workbooks are binary files, so GitHub cannot combine two people's changes.

1. Post in Discord before editing the workbook.
2. Only one person edits it at a time.
3. Pull the newest repository version before opening it.
4. Save and close Excel when finished.
5. Commit with a message such as `Update Sprint 2 week 1 hours`.
6. Push immediately and tell the group in Discord.

For simultaneous editing, upload the workbook to OneDrive and pin its sharing link
in Discord. The repository copy can then be updated at assignment checkpoints.
